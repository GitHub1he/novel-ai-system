## API 测试（可选）

### 使用 curl 测试

#### 1. 管理员获取所有项目
```bash
# 先登录获取 token
TOKEN=$(curl -X POST http://localhost:8000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username":"admin","password":"admin123"}' \
  | jq -r '.access_token')

# 获取项目列表
curl -X GET http://localhost:8000/api/projects/list \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json"

# 预期：返回所有项目
```

## 常见问题

### Q1: 登录后看不到项目列表？
**A:** 检查后端是否启动，查看浏览器控制台是否有错误

### Q2: 管理员看不到其他用户的项目？
**A:**
1. 确认使用的是 admin 账号（用户名：admin）
2. 检查数据库中是否有其他用户创建的项目
3. 查看后端日志，确认权限检查逻辑

### Q3: 普通用户能看到所有项目？
**A:**
1. 确认使用的是 testuser 账号（不是 admin）
2. 检查 JWT token 中的 is_admin 字段是否为 0
3. 查看后端日志，确认权限检查逻辑

### Q4: 无法访问管理面板？
**A:**
1. 确认使用 admin 账号登录
2. 检查前端路由配置
3. 查看浏览器控制台是否有错误
